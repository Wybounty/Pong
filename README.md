# 🏓 Pong

Un clone du jeu **Pong** développé avec **Unity** dans le but d'apprendre les bases du développement de jeux vidéo.

## 🎮 Fonctionnalités

- Déplacement des deux raquettes
- Mouvement de la balle
- Collisions avec les murs et les raquettes
- Système de score
- Remise en jeu après chaque point

## 🛠️ Technologies

- Unity 6
- C#
- TextMeshPro

## 🚀 Lancer le projet

1. Cloner le dépôt :

```bash
git clone https://github.com/Wybounty/Pong.git
```

2. Ouvrir le projet avec Unity Hub.
3. Lancer la scène principale.
